# CLOCK WEB APP

## HTML Part

### This is the structure of the webapplication

    * html:
    * head: contain the links
    * title: contain the title of the page
    * body: contain the whole page structure
    * div: divides the body into parts
    * class: used to identify a div
    * id: uniquely identify a div
    * button: used to display a button tag in the body
    *
## CSS Part 

### This is all about the styling of the page

    *  button:hover: this shows what happend when you place the cursor on the button

    * letter-spacing: The space between each letter

    * transform: The way it increases when its triggered

    *ransition: transform 0.2s; This shows the way it will increase or decrease in 0.2s
